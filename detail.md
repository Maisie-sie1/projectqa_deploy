## 🇬🇧 English Version (สำหรับ Resume ภาษาอังกฤษ)

**Lumina Books (Full-Stack E-Commerce Bookstore)**
- Developed a full-stack e-commerce bookstore web application using **Next.js 16 (App Router)**, **React 19**, and **TypeScript**
- Designed database schemas, relations, and optimized queries using **PostgreSQL** and **Drizzle ORM**
- Implemented secure session-based authentication and role-based access control (RBAC) via **JWT** and **HTTP-Only cookies**
- Created server-side logic using Next.js **Server Actions** integrated with **Zod** for robust data validation and form handling
- Built a real-time **Admin Dashboard** for sales metrics monitoring, bank slip verification, and inventory controls
- Developed automated End-to-End (E2E) testing suites using **Playwright** to ensure checkout and purchase flow reliability

## 🇹🇭 Thai Version (สำหรับ Resume ภาษาไทย)

**Lumina Books (ระบบร้านหนังสือออนไลน์แบบ Full-Stack)**
- พัฒนาเว็บแอปพลิเคชันระบบร้านขายหนังสือออนไลน์แบบ Full-stack ด้วย **Next.js 16 (App Router)**, **React 19** และ **TypeScript**
- ออกแบบโครงสร้างฐานข้อมูลเชิงสัมพันธ์ (Database Schema & Relations) และคำสั่งคิวรีข้อมูลด้วย **PostgreSQL** และ **Drizzle ORM**
- พัฒนาระบบยืนยันตัวตนและควบคุมสิทธิ์การเข้าใช้งาน (RBAC) ด้วย **JWT** บันทึกผ่าน **HTTP-Only Cookies** เพื่อความปลอดภัยสูง
- สร้างสถาปัตยกรรมจัดการข้อมูลฝั่งเซิร์ฟเวอร์ด้วย Next.js **Server Actions** และใช้ **Zod** ในการตรวจสอบความถูกต้องของข้อมูล
- สร้างแผงควบคุมหลังบ้าน (**Admin Dashboard**) สำหรับดูรายงานสถิติรายได้ ตรวจสอบสลิปการโอนเงิน และจัดการสต็อกหนังสือ
- เขียนระบบทดสอบการทำงานของระบบสั่งซื้อตั้งแต่เลือกหนังสือไปจนถึงการชำระเงินแบบอัตโนมัติ (E2E Testing) ด้วย **Playwright**
